# InDriver Bot - LSPosed Module

Автоматизация для inDriver:
- Автопринятие заказов (от 7000₸)
- Обход оплаты объявлений (2030₸ → 0₸)
- 25 бесплатных звонков
- Обход проверок (root, xposed, эмулятор)

## Сборка
APK автоматически собирается через GitHub Actions при пуше в main.

## Установка
1. Установите APK из Artifacts
2. Включите модуль в LSPosed
3. Выберите sinet.startup.indriver
4. Перезагрузите устройство
